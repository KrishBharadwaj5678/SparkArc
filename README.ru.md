# ✨ SparkArc

Завораживающий интерактивный **3D-эффект частиц**, созданный с использованием современных веб-технологий.

![SparkArcDemo](https://github.com/KrishBharadwaj5678/SparkArc/raw/main/SparkArcDemo.png)

## 🌟 Возможности

| Возможность                   | Описание                                                       |
| ----------------------------- | -------------------------------------------------------------- |
| 💫 **3D мир частиц**          | Тысячи светящихся кубов, плавающих в динамическом пространстве |
| 🔀 **Анимация перемешивания** | Одним кликом частицы меняют цвет                               |
| 🔥 **Эффекты свечения**       | Bloom-освещение и яркие неоновые цвета                         |
| 🖥️ **Работа в браузере**     | Плавно работает прямо в вебе                                   |
| 🎮 **Интерактивный опыт**     | Управляемые пользователем трансформации и движения             |

---

## 🛠️ Технологический стек

| Технология        | Назначение                                 |
| ----------------- | ------------------------------------------ |
| 🌐 **HTML**       | Основная структура приложения              |
| 🎨 **CSS**        | Стилизация, макет и дизайн интерфейса      |
| 🧠 **JavaScript** | Логика интерактивности и анимаций          |
| 🎮 **Three.js**   | 3D-движок для частиц и визуальных эффектов |
