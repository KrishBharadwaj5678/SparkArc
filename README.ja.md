# ✨ SparkArc

最新のウェブ技術で構築された、魅惑的なインタラクティブな **3Dパーティクル体験** です。

![SparkArcDemo](https://github.com/KrishBharadwaj5678/SparkArc/raw/main/SparkArcDemo.png)

## 🌟 特徴

| 機能                   | 説明                      |
| -------------------- | ----------------------- |
| 💫 **3Dパーティクルワールド**  | ダイナミックな空間に浮かぶ何千もの発光キューブ |
| 🔀 **シャッフルアニメーション**  | ワンクリックでパーティクルの色が変化      |
| 🔥 **没入感のある発光エフェクト** | ブルームライティングと鮮やかなネオンカラー   |
| 🖥️ **ブラウザベース**      | Web上でスムーズに動作            |
| 🎮 **インタラクティブ体験**    | ユーザー操作による変形や動き          |

---

## 🛠️ 技術スタック

| 技術                | 用途                           |
| ----------------- | ---------------------------- |
| 🌐 **HTML**       | アプリの基本構造                     |
| 🎨 **CSS**        | スタイリング・レイアウト・UIデザイン          |
| 🧠 **JavaScript** | インタラクションとアニメーションロジック         |
| 🎮 **Three.js**   | パーティクルとエフェクトのための3Dレンダリングエンジン |
