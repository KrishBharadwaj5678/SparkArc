# ✨ SparkArc

Uma experiência interativa hipnotizante de **partículas 3D**, construída com tecnologias web modernas.

![SparkArcDemo](https://github.com/KrishBharadwaj5678/SparkArc/raw/main/SparkArcDemo.png)

## 🌟 Funcionalidades

| Funcionalidade                     | Descrição                                                    |
| ---------------------------------- | ------------------------------------------------------------ |
| 💫 **Mundo de partículas 3D**      | Milhares de cubos brilhantes flutuando em um espaço dinâmico |
| 🔀 **Animação de embaralhamento**  | Com um clique, as partículas mudam de cor                    |
| 🔥 **Efeitos de brilho imersivos** | Iluminação bloom e cores neon vibrantes                      |
| 🖥️ **Baseado no navegador**       | Funciona suavemente na web                                   |
| 🎮 **Experiência interativa**      | Transformações e movimentos acionados pelo usuário           |

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia        | Finalidade                                         |
| ----------------- | -------------------------------------------------- |
| 🌐 **HTML**       | Estrutura principal do aplicativo                  |
| 🎨 **CSS**        | Estilização, layout e design da interface          |
| 🧠 **JavaScript** | Lógica de interatividade e animações               |
| 🎮 **Three.js**   | Motor de renderização 3D para partículas e efeitos |
